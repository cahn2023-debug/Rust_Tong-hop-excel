# Changelog: Excel Engine Ultra

## [v1.1.0] - 2026-04-07
### Added
- **3-Pane Dashboard:** Draggable sidebar and right panel for optimized screen usage.
- **Advanced Preview Table:** Linked-pair column resizing and Excel-like Sort & Filter functionality.
- **Export Folders:** Native OS integration to open output directories instantly.
- **Glassmorphism Components:** Updated filter menus and interactive button states.

## [v1.0.2] - 2026-04-07
### Added
- **Kinetic Blueprint UI:** Detailed technical design with mono fonts, coordinates, and pulses.
- **Frameless Implementation:** Custom title bar and window control integration.
- **Progress Tracking:** Real-time visual feedback for consolidation tasks.

## [v1.0.1] - 2026-04-07
### Added
- **SQLite Cache Layer:** Metadata tracking to skip redundant file parsing.
- **Tauri Integration:** Bridge between high-performance Rust core and modern UI.

## [v1.0.0] - 2026-04-07
### Added
- **Core Engine:** Parallel scanner and multi-threaded Excel parser using Rayon.
- **Aggregation System:** Thread-safe data merging via DashMap.
- **CLI Mode:** Initial test scripts for data generation and verification.
